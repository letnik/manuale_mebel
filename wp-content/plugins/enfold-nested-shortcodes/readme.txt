Enfold - Nested Shortcodes gives you the possibility to nest Enfold shortcodes in a very simple and comfortable way.

== Description ==
With this plugin you have completely new possibilities to redesign your Enfold pages and posts. Now it is very easy to nest enfold shortcodes into each other. This gives you a whole new way of configuring pages and content.

Enfold - Nested Shortcodes adds a new custom posttype. There you can easily create and manage your new shortcodes.

Then you can copy these shortcodes, or just paste them inside text fields using the tinyMCE button.

Please note: all shortcodes are stored in the database, but the are shipped with transients for a better performance.

== Installation ==
In order to install the plugin, you can easily upload it as a Zip-Archive in the WordPress Admin area, or simply upload the unpacked folder directory to your plugin folder via FTP.

Then you can activate it in the plugin directory.
